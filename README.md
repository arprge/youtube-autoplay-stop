# YouTube Autoplay Stop

A Tampermonkey userscript that prevents autoplay on YouTube's "Watch Later" playlist by pausing videos before they end.

## Installation

1. Install [Tampermonkey](https://www.tampermonkey.net/) browser extension
2. Open [scriptYT.js](scriptYT.js) and copy the script content
3. Create a new userscript in Tampermonkey and paste the code

## Features

- Automatically pauses videos 1 second before they end
- Only active on "Watch Later" playlist
- Prevents unwanted autoplay to the next video
